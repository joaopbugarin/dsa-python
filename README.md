# dsa-python

Progress tracker for working through the [NeetCode 150](https://neetcode.io/practice) in Python. Using it to improve my python skills and brush up algorithms.

## Structure

Problems are organised by NeetCode 150 topic. Each file is named after its LeetCode problem number. Redos are suffixed with `-r1`, `-r2`, etc. — when I revisit a problem later and redo it from scratch.
