# LeetCode Solutions in Python

This repository contains my solutions to various LeetCode challenges implemented in Python.

## Repository Structure

Each Python file with a number in its name corresponds to a specific LeetCode problem. The number in the filename matches the LeetCode problem ID.

For example:
- `123.py` - Solution to LeetCode Problem #123

## How to Run

Each solution file can be run independently:

```bash
python3 <problem_number>.py
```

## About LeetCode

[LeetCode](https://leetcode.com/) is a platform for preparing technical coding interviews. It features a vast collection of algorithmic problems that help developers enhance their coding skills and prepare for technical interviews.

## Progress

This repository is regularly updated with new solutions as I work through more LeetCode challenges.
